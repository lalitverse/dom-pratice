# DOM Manipulation Practice Sheet

This repository contains solutions for all 12 questions from the **Phase 6 DOM Manipulation Practice Sheet** using **HTML, CSS, and JavaScript**.

## 🟢 Easy Level

### Q1 - Change Heading Text
Change the text of an `<h1>` using JavaScript.

### Q2 - Change Background Color
Change the page background color on button click.

### Q3 - Display Input Value
Display user input inside a paragraph.

### Q4 - Hide and Show Text
Toggle paragraph visibility.

---

## 🟡 Moderate Level

### Q5 - Add New List Item
Create and append a new `<li>` element.

### Q6 - Remove a Card
Remove an entire card from the DOM.

### Q7 - Counter App
Increase and decrease a counter.

### Q8 - Live Character Counter
Count characters while typing.

---

## 🔴 Hard Level

### Q9 - Todo List
Add and display tasks dynamically.

### Q10 - Image Gallery
Display the next image on button click.

### Q11 - Dark Mode Toggle
Switch between light and dark themes.

### Q12 - Mini Student Manager
Add, display, and delete student cards.

## 🎯 Learning Outcomes

After completing this project, you will be able to:

- Select and manipulate DOM elements.
- Handle user events.
- Create and remove elements dynamically.
- Build small interactive applications.
- Manage application state using JavaScript.
